# 👥 HR Analytics Dashboard 📊

## 📌 Project Overview

This project focuses on analyzing employee data to understand **attrition trends, workforce distribution, and employee satisfaction** using interactive dashboards built in **Power BI**.

---

## 🎯 Objectives

* 📉 Analyze employee attrition
* 👨‍💼 Understand workforce demographics
* 😊 Measure job satisfaction
* 💰 Study salary and job role impact
* 📊 Provide actionable HR insights

---

## 🛠️ Tools & Technologies

* 📊 Power BI
* 📑 Excel / CSV Dataset
* 🔄 Power Query (Data Cleaning)
* 🧮 DAX (Data Analysis Expressions)

---

## 📊 Dashboard Features

* 📌 Attrition Analysis
* 👥 Employee Demographics
* 💼 Job Role Insights
* 💰 Salary Distribution
* 😊 Job Satisfaction Metrics

---

## 📈 Key Insights

* 🔴 High attrition in specific departments
* ⏱️ Overtime employees are more likely to leave
* 😊 Low job satisfaction leads to higher attrition
* 💼 Certain job roles have higher turnover

---

## 🚀 How to Use

1. 📥 Download the `.pbix` file
2. 📂 Open in Power BI Desktop
3. 🔄 Refresh data if needed
4. 🎛️ Explore interactive visuals
