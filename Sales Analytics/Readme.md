# 💰 Sales Analytics Dashboard 📊

## 📌 Project Overview

This project focuses on analyzing sales data to uncover insights about **revenue trends, product performance, and regional sales** using an interactive **Power BI dashboard**.

---

## 🎯 Objectives

* 📈 Analyze overall sales performance
* 🌍 Identify top-performing regions
* 🛍️ Understand product/category performance
* 📅 Track monthly and yearly trends
* 💡 Generate actionable business insights

---

## 🛠️ Tools & Technologies

* 📊 Power BI
* 📑 Excel / CSV Dataset
* 🔄 Power Query (Data Cleaning)
* 🧮 DAX (Data Analysis Expressions)

---

## 📊 Dashboard Features

* 📌 Revenue & Profit Analysis
* 🌍 Region-wise Sales Performance
* 🛍️ Category & Product Insights
* 📅 Time-based Trends (Monthly/Yearly)
* 📊 KPI Cards (Total Sales, Profit, Orders)

---

## 📈 Key Insights

* 📊 Certain regions contribute the highest revenue
* 🛍️ A few products dominate total sales
* 📅 Seasonal trends impact sales performance
