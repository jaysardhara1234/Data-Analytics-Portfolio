# 🎓 Student Performance Dashboard 📊

## 📌 Project Overview

This project analyzes student data to understand **academic performance, study habits, and factors affecting scores** using an interactive **Power BI dashboard**.

---

## 🎯 Objectives

* 📊 Analyze student performance across subjects
* 📚 Understand impact of study time on scores
* 🧾 Evaluate role of attendance in performance
* 👨‍🎓 Compare performance among different groups
* 💡 Generate insights to improve academic outcomes

---

## 🛠️ Tools & Technologies

* 📊 Power BI
* 📑 Excel / CSV Dataset
* 🔄 Power Query (Data Cleaning)
* 🧮 DAX (Data Analysis Expressions)

---

## 📊 Dashboard Features

* 📌 Overall Performance Analysis
* 📚 Study Time vs Score Insights
* 🧾 Attendance Impact Analysis
* 📊 Subject-wise Performance
* 📈 Score Distribution

---

## 📈 Key Insights

* 📚 Higher study time leads to better performance
* 🧾 Regular attendance improves scores
* 📊 Some subjects show lower average performance
* 👨‍🎓 Performance varies across student groups
